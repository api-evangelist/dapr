We are excited to announce v1.0.0-rc.3, the third release candidate for the official production ready v1.0 of Dapr! The improvements in this release came out of issues opened by the community which has helped test the previous release candidate as well as contributed code changes to address these issues. Once again, we encourage the Dapr community to upgrade to the this release candidate and identify problems and open issues.

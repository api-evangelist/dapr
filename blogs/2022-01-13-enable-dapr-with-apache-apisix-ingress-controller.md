Using Apache APISIX Ingress Controller with Dapr In this post we discuss the benefits of using Apache APISIX Ingress Controller with Dapr and describe a project between Weyhd and China Merchants International Technology that utilizes it. This post dives into how you can apply the Apache APISIX Ingress Controller with Dapr to your applications running in a Kubernetes cluster. Ingress is a resource that represents the entry point for traffic in a Kubernetes environment.
